## Usage

A toast notification you can drop into any scene to mark a moment — render complete, message delivered, deploy successful. Six anchor positions (`top-left`, `top-center`, `top-right`, `bottom-left`, `bottom-center`, `bottom-right`) place it against the edge of the canvas, and the slide direction adjusts automatically.

Pick a `variant` (`info`, `success`, `warning`, `error`) to change the icon and accent. `durationVisibleSec` controls how long the toast holds before it fades back out; tune it to match the surrounding scene length.
