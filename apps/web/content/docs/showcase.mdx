## Usage

Use Showcase to feature any other scene as the centerpiece of a hero shot — landing page reels, "what we built this quarter" decks, marketing pages. Pick the child component, a backdrop style (gradient, radial glow, dotted grid, or solid), and tune the frame size with `innerScale`.

The eyebrow / title / caption all stagger in independently before the framed child reveals, so the storytelling reads top-down before the eye lands on the demo inside.
